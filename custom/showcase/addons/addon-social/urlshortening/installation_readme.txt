Bitly requires an API key and user name to be registered.

You can obtain a user and api key from here: https://bitly.com/a/your_api_key

Then add the api key and username to your local.properties or project.properties like so :

bitly.apiKey=<api key>
bitly.username=<bitly user>

i.e.

bitly.apiKey=ldldld6dmvldldmddkdkdkd
bitly.username=mybitlyusername
